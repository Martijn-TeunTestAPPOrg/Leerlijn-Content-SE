## Opdracht 1
Zet de volgende casus om naar een While loop.

### Specificaties
* Maak een variabel `input` aan.
* Het programma blijft vragen om een invoer tot het juiste getal is geraden (bijvoorbeeld 7).
* Als het ingevoerde getal incorrect of correct is print dat dan in de console.

### Verwachte output:
Als `gok != juisteGetal`, dan komt in de console:
```
Raad het getal (tussen 1 en 10):
Helaas dat is niet correct.
Raad het getal (tussen 1 en 10):
```
Als `gok == juisteGetal`, dan komt in de console::
```
Gefeliciteerd! Dat is correct.
```

### Nu jij:
``` csharp runner
using System;

// Het juiste getal  
int correctNumber = 7;  
  
// Kies een random getal.  
Random rand = new Random();  
int number = rand.Next(5, 8);  
  
// While-lus om te blijven raden  
while ()  
{  
    //Kies een nieuw random getal  
    number = rand.Next(5, 8);  
}  
  
// Print wanneer getal correct is.
```

```
Voer een positief getal in:

    // Getal is ongeldig, kies een nieuw random getal.

{  
}

## Opdracht 3
Zet de volgende casus om naar een While loop.

### Specificaties
* Vraag de gebruiker om een reeks positieve getallen in te voeren.
* Tel de ingevoerde getallen bij elkaar op.
* Het programma stopt zodra de gebruiker "0" invoert.
* Toon de som van alle ingevoerde getallen.

### Verwachte output:
Als invoer bijvoorbeeld `5, 3, 2, 0` is:
```
Voer een positief getal in (0 om te stoppen):  
5  
Voer een positief getal in (0 om te stoppen):  
3  
Voer een positief getal in (0 om te stoppen):  
2  
Voer een positief getal in (0 om te stoppen):  
0  
Het totaal van alle ingevoerde getallen is: 10  
```

### Nu jij:
``` csharp runner
using System;

// Variabel om het totaal bij te houden  
int count = 0;  
  
// Kies een random getal  
Random rand = new Random();  
int number = rand.Next(0, 20);  
  
// While-lus om getallen te blijven vragen  
while ()  
{  

    // Kies een nieuw random getal  
    number = rand.Next(0, 20);  
}

// Print het totaal
```

> [!info]- Mogelijke uitwerking
> ``` csharp

>

>
>// Kies een random getal

>

>while (number != 0)

### Specificaties
* Vraag de gebruiker om een wachtwoord in te voeren.
* Het wachtwoord moet `C#Rocks` zijn.
* Het programma blijft vragen totdat het juiste wachtwoord is ingevoerd.
* Toon een succesbericht als het wachtwoord juist is

### Verwachte output:
Als invoer `123` is:
```
Voer het wachtwoord in:
Fout wachtwoord probeer opnieuw.
```
Als invoer `C#Rocks` is:
```
Voer het wachtwoord in:
Succes! Je bent ingelogd
```

### Nu jij:
``` csharp runner
using System;

// Random wachtwoorden  
string[] randomStrings = {"C#Rocks", "KotlinRocks", "JavaDoesntRock", "PythonRocks" };  
  
// Correcte wachtwoord  
string password = "C#Rocks";  
  
// Random gekozen wachtwoord  
string input = randomStrings[new Random().Next(randomStrings.Length)];  
  
// While-loop om het wachtwoord te blijven controleren.  
while(){  

	// Random gekozen wachtwoord is onjuist, kies een nieuwe
    input = randomStrings[new Random().Next(randomStrings.Length)];  
}

// Print als wachtwoord correct is
```

## Opdracht 5
Schrijf een programma dat de gebruiker vraagt om een getal in te voeren. Het programma vermenigvuldigt dat getal steeds met 2 totdat het resultaat groter is dan 100. Het programma moet de gebruiker ook laten zien hoeveel keer het getal is vermenigvuldigd voordat het resultaat groter is dan 100.

### Specificaties
* Vraag de gebruiker om een getal in te voeren
* Vermenigvuldig het getal met `2` tot het resultaat groter is dan `100`
* Toon hoe vaak het getal is vermenigvuldigd

### Verwachte output:
Als de invoer 5 is:
```
Het getal is 7 keer vermenigvuldigd totdat het groter was dan 100.
```
Als de invoer 10 is:
```
Het getal is 6 keer vermenigvuldigd totdat het groter was dan 100.
```

### Nu jij:
``` csharp runner
using System;

// Variabele om aantal loops bij te houden.
int count = 0;  
  
//Kies een random getal  
Random rand = new Random();  
int number = rand.Next(0, 100);  

// Toon het gekozen getal
Console.WriteLine($"Getal is: {number}");  
  
do  
{
} while ();

// Print aantal keer dat getal is vermenigvuldigd
```

> [!info]- Mogelijke uitwerking
> ``` csharp
> using System;
> 
>// Initializeer het getal dat bijhoudt hoe vaak de while-loop is uitgevoerd.
>int count = 0;
>
>//Kies een random getal
>Random rand = new Random();
>int number = rand.Next(0, 100);
>        
>// Toon het gekozen getal
>Console.WriteLine($"Getal is: {number}");
>
>// Getal x2 tot het groter is dan 100 en bijhouden van aantal loopjes.
>do
>{
>	number *= 2;
>	count++;
>} while (number <= 100);
>
>// Print aantal keer dat getal is vermenigvuldigd
>Console.WriteLine($"Het getal is {count} keer vermenigvuldigd totdat het groter was dan 100.");
>```
